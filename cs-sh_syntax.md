# Computer Simulator Shell Language

___

## Variables

```@var = 10```

- `@` is the variable declarer, it signals that a variable is being created.
    - If no value is assigned and the variable
      exists then the shell will print the value in the variable.
- `var` this is the identifier that is used to access the variable at later points.
- `=` is the assignment operator.
- Currently, Variables support holding *string* and *numerical* datatypes

## Functions

```func(var, 10)```

**NOTE:** Not currently possible to define a function

- `func` the function identifier
